## 2\. 核心使命與願景 (Mission & Vision)

### 🌍 My Vision (我們眼中的未來)

**"A world connected by deep, meaningful relationships."** 我們希望創造一個世界，在那裡，每個人都能因為擁有深厚、真實的人際連結，而活得更快樂、健康且充滿目標感。

### 🚀 My Mission (我們每天在做的事)

**"To build the digital infrastructure for emotional connection."** 我們利用科技（AI + 軟體），消除人際溝通中的誤解與隔閡，幫助人們在忙碌的現代生活中，輕鬆地維護並增強與伴侶、家人及朋友的深度連結。

### 🎯 The Goal (具體目標)

打造一款 **「關係健身房 (Relationship Gym)」** 應用程式。就像去健身房鍛鍊肌肉一樣，讓人們透過每天的小行動與深度對話，持續鍛鍊「關係肌力」。

## 4\. 產品藍圖：我們如何打造？ (The Blueprint)

我們將打造一個 **訂閱制 (SaaS)** 的情感健康平台，核心公式為：

> **Haven = AI Intelligence + Psychological Frameworks + Gamified Interaction**

### 🚀 Phase 1: Haven for Couples (伴侶關係健身房)

**Target Audience:** 渴望深化關係、解決溝通障礙的伴侶。

#### A. The Infinite Deck (無限牌組系統)

*Powered by Generative AI & Psychology*

1. **Daily Vibe (日常連結)**：輕鬆的破冰話題，分享今日心情。

2. **Soul Dive (靈魂深潛)**：觸及價值觀、夢想與恐懼的深度對話。

3. **Safe Zone (衝突修復)**：當爭吵發生時，引導雙方冷靜、表達需求而非情緒。

4. **After Dark (親密升溫)**：探索慾望與親密關係的情趣引導。

5. **Co-Pilot (關係同步)**：每週/每月的關係覆盤會議 (Relationship Review)。

6. **Love Blueprint (訂製卡牌)**：針對特定情境（如遠距離、備孕中）生成的專屬卡牌。

#### B. The AI Mediator (AI 情感中介系統)

- **AI Love Language Translator (愛之語翻譯機)**：

   - **Input**: User 的碎念、日記或抱怨。

   - **Output**:

      1. **情緒解碼**: "你現在感到焦慮，是因為..."

      2. **深層需求**: "你其實不是在生氣他不洗碗，而是需要被重視。"

      3. **給 User 的建議**：給 User 提供一些建議、行動建議以及可以反思的地方、幫助 User 反思的提問等等。

      4. **給 User 的伴侶的建議與行動指南**: 給伴侶的一封「說明書」，告訴對方如何正確安慰你。

- **Relationship Check-up (關係健檢)**：基於互動數據，評估關係健康度並提供「維他命（建議行動）」。

#### C. Daily Action (每日微行動)

- 簡單、低門檻的互動挑戰 (e.g., "現在傳一張你覺得最像他的動物照片給他").
